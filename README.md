# Angular Todo App

A modern, polished todo management experience built with Angular. The interface blends gradient-backed cards, contextual filters, and expressive affordances so you can prioritize work quickly and confidently.

## Features

- **Priority smart list:** Tag every task with `High`, `Medium`, or `Low` priority and see the badge color change accordingly.
- **Filtered workspace:** Switch between all tasks, what’s next, and completed items with a single tap.
- **Composed inputs:** Add notes, due dates, and richly styled metadata for each task before committing it to your list.
- **Persistent stats:** Track total, active, and complete tasks along with a progress gauge that animates with every update.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:4200/` in your browser. The server reloads automatically when you change source files.

## Building for production

```bash
npm run build
```

Production artifacts are generated inside the `dist/` directory.

## Testing

- Run unit tests:
  ```bash
  npm run test
  ```

- Run end-to-end tests (if needed):
  ```bash
  npm run e2e
  ```
